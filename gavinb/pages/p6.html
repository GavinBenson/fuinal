Crossword RegEx

Answer: NOSFERATU

  1. 1st row, 1st column it has to be part of the [ONE] character set and be either NF, FA, A, or FN going down.
  2. 1st row, 2nd column it has to be either P, ET, O, or TEA going down. A part of the [ONE] character set, and be either an O or S.
  3. 1st row, 3rd column it has to fit the chracter sets of [RUSH],[SUPER],[SKA], and be either an O or S.
  4. 2nd row, 1st column is fulfilled by NF going down from the first row.
  5. 2nd row, 2nd column has to either be 'RE' or 'ER' and fulfill either P, ET, O, or TEA.
  6. 2nd row, 3rd column is determined by 2nd row, 2nd column.
  7. 3rd row, 1st column has to start with A, due to the fact that 'A+' means at least one or more A and it is at the beginning.
  8. 3rd row, 2nd column is fulfilled by 2nd row, 2nd column.
  9. 3rd row, 3rd column has to be part of the character sets of [RUSH],[STUPA],[SUPER], and [TUB].
